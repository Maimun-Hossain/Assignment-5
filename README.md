1️⃣ What is the difference between var, let, and const?

Ans: var can be updated and redeclared anytime, let can be updated anytime but it can not be redeclared, const can not be redeclared and updated but it must be initialized when declared.

2️⃣ What is the spread operator (...)?

Ans: spread operator can spreads elements of an arrays or objects into individual elements which can make it easier to use of data.

3️⃣ What is the difference between map(), filter(), and forEach()?

Ans: map() can create a new array to each element, filter() can create a new array with the elements that matches condition and forEach() it runs a function on each element and does not return a new array.
